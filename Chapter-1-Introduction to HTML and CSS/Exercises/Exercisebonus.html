HTML & CSS Quick-Reference Guide

Welcome to our quick-reference guide covering essential HTML tags and CSS properties.

HTML Tags

  • <div>: Defines a division or section in an HTML document. 
    <div>
    Content goes here
</div>
  • <p>: Defines a paragraph. 
    <p>This is a paragraph</p>
  • <img>: Embeds an image. 
    <img src="image.jpg" alt="Description">
  • <li>: Tag defines a list item
